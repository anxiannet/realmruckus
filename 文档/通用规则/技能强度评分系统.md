# 通用技能强度评分系统

生效日期：2026-07-14

## 一、文件定位

本文件只迁移当前源评分标准中已经存在的通用技能强度评分方法。

技能强度评分：

- 只评价规则强度；
- 使用整数；
- 采用无上限累计分；
- 不直接决定卡牌等级；
- 不替代测试和正式规则文本；
- 不评价具体世界观符合度。

## 二、核心公式

```text
技能强度评分
= 锚点基础分
+ 受影响战力修正
+ 数量修正
+ 持续时间修正
+ 区域连锁修正
+ 行动节奏修正
+ 资源差修正
+ 选择自由度修正
+ 多玩家 / 多区域修正
- 等级限制
- 前置条件
- 使用成本
- 反制风险
```

推荐简式：

```text
技能强度评分
= 锚点分
+ 超出锚点的放大因子
- 限制因子
```

同一收益不得重复计算。

## 三、强度区间

| 分数范围 | 强度层级 |
|---:|---|
| 0 | 无效 |
| 1–19 | 极弱 |
| 20–39 | 弱 |
| 40–59 | 中 |
| 60–79 | 强 |
| 80–99 | 极强 |
| 100–149 | 超强 |
| 150–219 | 破局级 |
| 220以上 | 失控级 / Boss级 |

100是强技能参照点，不是满分。

## 四、锚点优先原则

先选择最接近的锚点。

若锚点已经包含对象、数量、区域无主、区域回流、战力总量和行动布局影响，不得重复加分。

只能对锚点之外的新效果继续修正。

| 因子 | 扣分 |
|---|---:|

| 需要指定分组或指定区域 | -14 |
