# 通用规则状态机

规则版本：V1.3  
更新时间：2026-07-14

## 一、文件定位

本文件只把当前核心规则的执行顺序转写为状态流程，不新增状态字段、响应层级或引擎功能。

## 二、回合流程

```text
TURN_START
→ DRAW
→ CHOOSE_MAIN_ACTION_OR_PASS
→ RESOLVE_ACTION
→ RESOLVE_FORCED_EFFECTS
→ RECYCLE_EMPTY_AREAS
→ REFILL_CENTER_AREAS
→ CHECK_IMMEDIATE_WIN
→ HAND_LIMIT
→ CHECK_DECK_END
→ TURN_END
```

公共牌库为空时跳过 `DRAW`，其余流程不变。

任何效果产生空的已控制区域时，立即完成回流和中央区域补充，再继续后续结算。

## 三、部署流程

```text
SELECT_UNIT_FROM_HAND
→ SELECT_LEGAL_AREA
→ MOVE_UNIT_TO_AREA
→ REFILL_CENTER_IF_NEEDED
→ RESOLVE_ENTER_EFFECT
→ RECYCLE_EMPTY_AREAS
→ REFILL_CENTER_AREAS
→ CHECK_IMMEDIATE_WIN
```

合法区域：

- 中央公开无主区域；或
- 自己控制且未达到驻军上限的区域。

额外部署的单位也按相同流程结算入场效果。

## 四、进攻流程

```text
SELECT_ORIGIN
→ SELECT_OTHER_PLAYER_TARGET
→ VALIDATE_TARGET_PROTECTION
→ SELECT_1_TO_3_ATTACKERS
→ ORDER_ATTACKERS
→ RESOLVE_DUELS
→ RESOLVE_ATTACK_RESULT
→ RECYCLE_EMPTY_AREAS
→ REFILL_CENTER_AREAS
→ RESTORE_SURVIVORS
→ CHECK_IMMEDIATE_WIN
```

不能攻击：

- 自己控制的区域；
- 中央无主区域；
- 受到禁止进攻保护的区域。

## 五、效果牌流程

```text
SELECT_EFFECT_CARD
→ VALIDATE_COST_AND_TARGETS
→ PLAY_CARD
→ PAY_REQUIRED_COST
→ OPEN_LEGAL_RESPONSE_WINDOW
→ RESOLVE_OR_CANCEL
→ MOVE_CARD_TO_RULE_DEFINED_DESTINATION
→ RECYCLE_EMPTY_AREAS
→ REFILL_CENTER_AREAS
→ CHECK_IMMEDIATE_WIN
```

- 没有合法目标或无法支付费用时，不能发动。
- 被取消的效果不发生。
- 被取消的效果牌与取消牌按当前规则进入弃牌堆。
- 持续类效果牌按其正式说明留在场上，结束时进入弃牌堆。
- 当前基础规则不使用复杂反制连锁。

## 六、单次对决

```text
attacker_damage_taken += defender.level
defender_damage_taken += attacker.level
```

双方伤害同时生效。

单位累计伤害达到或超过自身等级时立即被击败。

防守单位伤害在本次战斗中累计；每个进攻单位最多攻击1次。

## 七、进攻结果

成功条件：

```text
defender_survivors == 0
and attacker_survivors >= 1
```

成功后：

- 存活进攻单位进入目标区域；
- 进攻玩家获得控制权；
- 目标区域原保护状态失效，除非保护规则另有规定；
- 出兵区域为空时立即回流。

失败后：

- 存活进攻单位返回出兵区域；
- 存活防守单位留在目标区域；
- 控制权不变。

双方全灭时，目标区域回流，不构成进攻成功。

## 八、区域回流

```text
IF controlled_area.units == 0
THEN
  remove_control
  end_area_attached_effects
  move_area_to_area_deck
  shuffle_area_deck
  refill_center_to_configured_size
```

回流检查持续到没有空的已控制区域。

附着在该区域并规定“区域变为无主时结束”的持续效果同时结束，并按规则进入弃牌堆。

## 九、即时胜利

任何控制权变化和主要行动完整结算后检查：

```text
count controlled areas by group
if any group_count == 3:
    end_game_immediately
```

即时胜利不等待回合或整轮结束。

## 十、牌库耗尽

公共牌库为空后，每个回合仍正常执行主要行动。

若连续完整一轮：

```text
所有玩家均未打出单位牌或效果牌
and
没有区域控制权变化
```

则按当前核心规则进入最终比较。

## 十一、必须阻止的非法状态

- 同一区域出现多个玩家的驻军；
- 超过驻军上限；
- 从手牌直接发起进攻；
- 攻击中央无主区域；
- 攻击自己的区域；
- 攻击受保护且禁止进攻的区域；
- 使用没有合法目标且无法结算的行动；
- 已回流区域仍保留控制权或附着持续效果；
- 被击败单位仍留在区域；
- 胜利确定后继续执行非必要状态变化。
