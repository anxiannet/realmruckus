# 通用规则层

规则版本：V1.3  
更新时间：2026-07-14

## 文件定位

本目录是项目当前通用机制、规则、卡表、评分模型、数学模型、测试规范和验证结果的唯一来源。

本层只保留当前最新有效内容，不保存：

- 旧规则版本
- 历史候选方案
- 被替代的参数
- 旧测试结果
- 具体卡名
- 角色或世界观名称
- 具体品牌卡号
- 包装或美术表达

多个具体产品表达均通过映射引用本通用规则层。

## 主从关系

```text
通用规则层
├── 具体表达A
├── 具体表达B
└── 后续具体表达
```

具体表达只能替换名称、世界观、插画、语言、UI、包装和卡面编号前缀，不得修改通用机制、参数与评分公式。

## 唯一数据源

```text
data/core_set.json
data/expansion_1.json
```

Markdown卡表只作人类可读说明，若与JSON冲突，以JSON为准。

## 维护原则

- 只维护当前最新有效内容。
- 新版本生效后直接替换旧内容，不在通用层并列保留历史版本。
- 通用机制只在本层维护。
- 具体表达不得反向写入通用层。
- 三套评分系统必须独立维护。
- 通用规则变化必须同步更新卡表、测试规范和当前测试基线。
- 来源记录只用于说明当前内容的抽取来源，不保存旧版本正文。
