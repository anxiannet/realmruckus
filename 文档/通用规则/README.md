# 通用规则层

规则版本：V1.3  
更新时间：2026-07-14

## 文件定位

本目录是项目的通用机制、规则、数学模型、测试规范和验证结果的唯一来源。

本层不包含：

- 具体卡名
- 角色名
- 世界观名称
- 阵营名
- 地名
- 皮肤专属术语
- 包装或美术表达

《夕妖：抢地盘》和 `Realm Ruckus` 均为本通用规则层的具体表达。

## 主从关系

```text
通用规则层（本目录）
├── 表达A：《夕妖：抢地盘》
└── 表达B：Realm Ruckus
```

具体表达只能替换：

- 名称
- 世界观
- 插画
- 语言
- UI
- 包装
- 卡面编号前缀

具体表达不得自行修改：

- 核心胜利条件
- 回合结构
- 战斗结算
- 区域回流
- 手牌上限
- 驻军上限
- 牌库耗尽结算
- 已锁定的组件模型

## 维护原则

- 通用机制只在本目录维护。
- 具体表达文件通过映射引用本层，不复制整套核心规则。
- 通用规则变更必须更新测试代码和历史基线说明。
- 具体表达专属内容不得反向写入通用规则层。
