# 通用核心机制

规则版本：V1.3  
更新时间：2026-07-14

## 一、游戏定位

- 玩家人数：2–4人
- 目标时长：15–30分钟
- 类型：轻策略、强互动、共享牌库、区域控制
- 核心目标：控制同一分组中的全部3个区域，立即获胜

## 二、标准参数

| 参数 | 标准值 |
|---|---:|
| 起始手牌 | 5 |
| 手牌上限 | 7 |
| 公共牌库 | 40张 |
| 区域牌库 | 12张 |
| 区域分组 | 4组 |
| 每组区域 | 3张 |
| 中央公开区域 | 3张 |
| 每个区域驻军上限 | 3个单位 |

## 三、组件类型

### 单位牌

- 可以从手牌部署到区域。
- 进入区域后成为驻军。
- 具有等级值。
- 等级值同时表示战斗生命与伤害。

### 效果牌

- 作为主要行动使用，除非文字明确允许反制或额外使用。
- 不具有等级值。
- 不成为驻军。

### 区域牌

- 构成独立牌库。
- 每3张属于同一分组。
- 控制同一分组全部3张即立即获胜。

## 四、开局

1. 洗混单位牌与效果牌，组成公共牌库。
2. 每名玩家抽5张起始手牌。
3. 洗混区域牌库。
4. 翻开指定数量的中央公开区域。
5. 随机决定先手。
6. 顺时针进行回合。

开局时没有玩家控制区域。

## 五、回合流程

1. 抽牌阶段：从公共牌库抽1张。
2. 主要行动阶段：选择1个主要行动，或不行动。
3. 结算阶段：完整结算行动、效果或战斗。
4. 结束阶段：若手牌超过上限，弃至上限。
5. 检查即时胜利与牌库耗尽结算条件。

公共牌库为空时跳过抽牌，其他阶段照常进行。

## 六、主要行动

### 部署

从手牌打出1张单位牌，放入：

- 1个中央公开的无主区域；或
- 自己控制且未达到驻军上限的区域。

部署完成后结算入场效果，再检查区域回流和胜利。

### 进攻

1. 选择自己控制且至少有1个驻军的区域作为出兵区。
2. 选择其他玩家控制且至少有1个驻军的区域作为目标区。
3. 从出兵区选择1–3个驻军作为进攻单位。
4. 依次结算每个进攻单位的攻击。
5. 根据结果处理控制权、存活单位和区域回流。

不能攻击自己控制的区域，也不能直接攻击中央无主区域。

### 使用效果牌

1. 从手牌选择1张效果牌。
2. 确认合法目标与发动条件。
3. 打出并选择目标。
4. 处理合法反制。
5. 完整结算效果。
6. 处理区域回流和胜利检查。

## 七、战斗

- 每个单位独立战斗，不按双方总等级直接相减。
- 进攻方决定攻击顺序。
- 每个进攻单位最多攻击1次。
- 每次攻击选择1个仍存活的防守单位。
- 双方同时造成等于自身等级的伤害。
- 累计伤害达到或超过自身等级时，该单位被击败。
- 防守单位伤害在本次战斗中累计。
- 战斗结束后，所有存活单位恢复至完整状态。

### 进攻成功

满足：

- 目标区没有防守单位存活；且
- 至少有1个进攻单位存活。

结果：

- 存活进攻单位进入目标区。
- 进攻玩家获得目标区控制权。
- 检查出兵区是否为空。
- 检查即时胜利。

### 进攻失败

若所有进攻单位均完成攻击后，目标区仍有防守单位存活：

- 存活进攻单位返回出兵区。
- 存活防守单位留在目标区。
- 控制权不变。

若双方全部被击败，目标区回流，进攻方不获得该区域。

## 八、区域控制与回流

- 区域至少有1个驻军时，由驻军所属玩家控制。
- 同一区域不能由不同玩家共同驻守。
- 区域失去最后1个驻军时立即变为无主。
- 将该区域放回区域牌库并重洗。
- 若中央公开区域不足指定数量，立即补充。

回流可能由战斗、返回手牌、击败、转移或其他效果触发。

## 九、通用结算原则

- 卡牌效果按正式文本精确执行。
- 只有明确写出“最多”“可以”“选择”时才能减少数量或不执行。
- 没有合法目标时，效果不结算。
- 费用必须先支付；效果被取消时费用通常不退还。
- 入场效果在单位正常成为驻军后结算。
- 任何效果使区域失去最后驻军时，立即执行回流。
- 具体卡牌文本与通用规则冲突时，正式卡牌文本优先。

## 十、牌库耗尽

公共牌库耗尽后不重洗弃牌堆。

若连续完整一轮同时满足：

1. 没有玩家打出单位牌或效果牌；
2. 没有区域控制权变化；

则进入最终比较：

1. 控制区域数量；
2. 场上驻军等级总和；
3. 单一分组中控制的最多区域数；
4. 场上驻军数量；
5. 仍相同则平局。

任何时刻满足同组3区控制，仍然立即获胜。
