# 通用核心机制

规则版本：V1.3  
更新时间：2026-07-14

## 一、游戏定位

- 玩家人数：2–4人
- 目标时长：15–30分钟
- 类型：轻策略、强互动、共享牌库、区域控制
- 核心目标：控制同一分组中的全部3个区域，立即获胜

## 二、标准参数

| 参数 | 标准值 |
|---|---:|
| 起始手牌 | 5 |
| 手牌上限 | 7 |
| 公共牌库 | 40张 |
| 区域牌库 | 12张 |
| 区域分组 | 4组 |
| 每组区域 | 3张 |
| 中央公开区域 | 3张 |
| 每个区域驻军上限 | 3个单位 |

## 三、组件类型

### 单位牌

- 从手牌部署到区域后成为驻军。
- 具有等级值。
- 等级值同时表示战斗生命与伤害。

### 效果牌

- 作为主要行动使用，除非文字明确规定反制或额外使用方式。
- 不具有等级值，不成为驻军。
- 正常结算完毕后进入弃牌堆；持续类效果按对应说明放置。

### 区域牌

- 构成独立牌库，不进入公共牌弃牌堆。
- 每3张属于同一分组。
- 控制同一分组全部3张即立即获胜。

## 四、开局

1. 洗混单位牌与效果牌，组成公共牌库。
2. 每名玩家抽5张起始手牌。
3. 洗混12张区域牌。
4. 翻开3张中央公开区域。
5. 随机决定先手。
6. 顺时针进行回合。

开局时没有玩家控制区域。

## 五、中央区域

- 标准状态尽量保持3个公开无主区域。
- 只能通过部署占领中央无主区域，不能直接进攻。
- 中央区域被占领后立即从区域牌库补充。
- 区域牌库为空时不再补充。

## 六、回合流程

1. 抽牌阶段：从公共牌库抽1张。
2. 主要行动阶段：选择1个主要行动，也可以不行动。
3. 结算阶段：完整结算行动、效果或战斗。
4. 结束阶段：若手牌超过7张，由当前玩家自行弃至7张。
5. 检查即时胜利与牌库耗尽结算条件。

公共牌库为空时跳过抽牌，其他阶段照常进行。

每回合最多执行1个主要行动；明确写出的额外行动和反制不受此限制。

## 七、主要行动

### 部署

1. 从手牌选择1张单位牌。
2. 选择中央公开无主区域，或自己控制且未满的区域。
3. 单位进入区域成为驻军。
4. 结算入场效果。
5. 处理回流、抽弃牌及其他后续效果。
6. 检查胜利条件。

单位不能从手牌直接发动进攻。

### 进攻

1. 选择自己控制且至少有1个驻军的区域作为出兵区。
2. 选择另一玩家控制且至少有1个驻军的区域作为目标区。
3. 选择1–3个驻军作为进攻单位。
4. 未参战单位留在出兵区。
5. 按战斗流程依次结算。
6. 处理控制权、存活单位和区域回流。
7. 所有存活单位恢复至完整状态。
8. 检查胜利条件。

不能攻击自己控制的区域或中央无主区域。

若目标区域受到禁止进攻的保护，则不是合法目标。

### 使用效果牌

1. 从手牌选择1张效果牌。
2. 确认合法目标和发动条件。
3. 打出并选择目标。
4. 处理合法反制。
5. 未被取消时完整结算效果。
6. 结算完毕后进入弃牌堆；持续类效果按对应说明放置。
7. 处理区域回流和其他后续效果。
8. 检查胜利条件。

反制效果不占使用者自己回合的主要行动，但必须在规定时机使用。

被反制取消的效果牌视为已使用，其效果不发生，并按对应规则处理去向。

## 八、战斗流程

- 每个单位独立战斗，不按双方等级总和直接相减。
- 进攻方决定攻击顺序。
- 每个进攻单位最多攻击1次。
- 每次攻击选择1个仍存活的防守单位。
- 双方同时造成等于自身等级的伤害。
- 即使一方被本次伤害击败，双方伤害仍同时生效。
- 累计伤害达到或超过等级时，单位立即被击败并移出区域。
- 被击败单位进入弃牌堆，除非具有替代结算。
- 防守单位伤害在本次战斗中累计。
- 目标区已无防守单位时，不再进行后续攻击。
- 战斗结束后所有存活单位恢复至完整状态。

### 进攻成功

条件：

- 目标区没有防守单位存活；且
- 至少有1个进攻单位存活。

结果：

1. 原控制者失去目标区控制权。
2. 存活进攻单位进入目标区。
3. 进攻玩家获得控制权。
4. 目标区原有保护状态失效，除非保护效果另有规定。
5. 出兵区为空时立即回流。
6. 检查胜利条件。

### 进攻失败

若所有进攻单位完成攻击后仍有防守单位存活：

1. 存活进攻单位返回出兵区。
2. 存活防守单位留在目标区。
3. 已被击败单位不返回。
4. 控制权不变。

若双方全部被击败，目标区回流，进攻方不获得该区域。

## 九、区域控制与回流

- 区域至少有1个驻军时，由驻军所属玩家控制。
- 同一区域不能由不同玩家共同驻守。
- 区域失去最后1个驻军时立即变为无主。
- 将该区域放回区域牌库并重洗。
- 若中央公开区域不足指定数量，立即补充至指定数量或牌库为空。

回流可能由战斗、返回手牌、击败、转移或其他效果触发。

## 十、通用结算原则

- 卡牌效果按当前正式文本精确执行。
- 目标、持续时间、触发条件与特殊边界以当前卡表及对应正式说明为准。
- 只有明确写出“最多”“至多”“可选择”或“或”时，才能减少数量或选择不执行。
- 没有可选表述时，应尽可能完整执行规定数量和范围。
- 没有合法目标时，效果不结算。
- 发动费用必须先支付；支付后即使被取消或目标失效，费用通常不退还。
- 入场效果在单位正常成为驻军后结算。
- 额外部署的单位也正常结算入场效果。
- 效果使区域失去最后驻军时，立即回流，再继续后续步骤。
- 卡牌效果与核心规则冲突时，当前正式卡牌说明优先。

## 十一、通用术语

### 返回手牌

- 将目标驻军放回其拥有者手牌。
- 不视为击败，不进入弃牌堆，不触发被击败效果。
- 移走最后驻军时触发区域回流。

### 击败

- 将目标驻军放入弃牌堆。
- 若具有替代结算，则按替代规则处理。

### 加入使用者手牌

- 将指定卡牌移入效果使用者手牌，并变更拥有者。

### 加入使用者区域

- 将指定驻军移入效果使用者控制且有空位的合法区域，并变更为该玩家的驻军。
- 没有可容纳的合法区域时，没有合法目标。

### 随机

- 从所有符合条件的对象中，通过洗牌、掷骰或其他双方认可方式决定结果。

### 暗选

- 各玩家在不展示的情况下同时选择，确认后同时公开或交换。

### 保护

- 受保护区域或驻军不能成为保护效果所禁止的进攻或效果目标。
- 具体范围、持续时间和能否被反制，以对应正式说明为准。

## 十二、牌库耗尽

公共牌库耗尽后不重洗弃牌堆。

玩家仍可使用手牌、部署、使用效果牌和发动进攻，并继续检查即时胜利。

若连续完整一轮同时满足：

1. 所有玩家均未打出单位牌或效果牌；
2. 没有任何区域控制权变化；

则按以下顺序比较：

1. 控制区域数量；
2. 场上驻军等级总和；
3. 单一分组中控制的最多区域数；
4. 场上驻军数量；
5. 仍相同则平局。

“一整轮”表示从某名玩家回合开始，直到所有玩家都完成1个回合。

任何时刻满足同组3区控制，仍然立即获胜。
