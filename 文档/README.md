## 数据维护原则

- JSON是机制卡表唯一数据来源。
- Markdown卡表只作人类可读说明。
- 机制修改必须先改JSON，再同步说明文档和具体表达。
- 引擎与模拟器不得通过具体卡名判断效果。

## 进度维护

每次完成规则、代码、测试或具体表达任务后，必须更新 `当前项目进度.md`。
