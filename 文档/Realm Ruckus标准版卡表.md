Applicable Rules Version: V1.3

## Document Status

This file is the production mapping for the Realm Ruckus skin.

It does not replace `规则/V1.3-基础卡表.md`. Quantities, Stars, effect boundaries and rules mechanics remain controlled by the Chinese source card list.

Formal RRK card codes have not yet been assigned. The `RRK Code` field remains `TBD` until the name, English card text, illustration and layout of each unique card face are approved.

## RRK Code Assignment Gate

Assign an RRK card code only after all of the following are approved for that card face:

1. Final English name.
2. Final English card text.
3. Final Stars and type confirmed against the source card list.
4. Final illustration.
5. Final frame, icons and layout.
6. Language and proofreading approval.
7. Print-ready card face approval.

Identical printed copies share one RRK card code. Any printed-content change requires a new RRK card code.
