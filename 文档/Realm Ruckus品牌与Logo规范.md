# Realm Ruckus 品牌与Logo规范

更新时间：2026-07-14  
状态：品牌候选阶段，正式商标与最终Logo尚未锁定

## 一、品牌结构

主品牌：

```text
REALM RUCKUS
```

宣传语：

```text
Monsters. Mischief. Total Takeover.
```

基础版本标识：

```text
Core Set
```

未来扩展使用：

```text
Realm Ruckus: Expansion Name
```

不得让扩展名称取代主品牌。

## 二、品牌语气

品牌表达应当：

- 顽皮
- 热闹
- 直接
- 易懂
- 有怪物个性
- 竞争但不凶狠

避免：

- 严肃史诗腔
- 过度幼儿化
- 血腥、恐怖或末日表达
- 难懂的伪古英语
- 现实族群、口音或文化刻板印象

## 三、Logo构成

建议正式Logo包含：

1. `REALM RUCKUS`英文文字主体
2. 粗厚奇幻材质
3. R字母押头韵带来的整体节奏
4. 可选的角、兽牙、石块或徽章装饰
5. 独立于背景的清楚外轮廓

Logo不得：

- 直接复制现有游戏Logo结构
- 使用难以辨认的花体
- 让装饰穿过关键字母
- 把宣传语永久焊死在Logo主体中
- 依赖复杂背景才能识别

## 四、Logo版本

最终应交付：

```text
主彩色Logo
单色黑Logo
单色白Logo
横版Logo
紧凑堆叠Logo
透明背景PNG
矢量SVG或PDF
小尺寸图标版
```

宣传语作为独立可选层维护。

## 五、安全区

Logo四周至少保留相当于主标题字母 `R` 高度25%的空白。

安全区内不得放置：

- 卡牌信息
- 人物脸部
- 条码
- 其他品牌Logo
- 高对比装饰

## 六、最小尺寸

正式尺寸需在最终矢量Logo完成后测试。

初步要求：

- 印刷Logo缩小后仍能清楚读出两个单词。
- 宣传语在无法清楚阅读时应移除，而不是继续缩小。
- 社交头像或小图标使用简化徽章版，不强塞完整宣传语。

## 七、颜色方向

当前视觉锚点中的建议方向：

- `REALM`：温暖金黄、石材或木石质感
- `RUCKUS`：高识别紫色、魔法石材质感
- 外轮廓：深灰、黑铁或深蓝黑

正式色值必须在CMYK软打样和实体打样后锁定。

不得仅按屏幕RGB颜色直接定义品牌印刷色。

## 八、背景适配

Logo必须能够用于：

- 明亮背景
- 深色背景
- 复杂插画背景
- 单色包装侧面
- 卡背
- 指南卡
- 社交媒体头像与横幅

复杂背景上必须使用描边、底板或阴影确保可读性。

## 九、TM与版权标识

- 品牌候选阶段可以根据策略使用 `TM`。
- 未取得注册前不得使用 `®`。
- TM只表示主张品牌使用，不代表注册成功。
- TM的位置和比例应固定，不随意放大。
- 版权标识与商标标识分开处理。

## 十、命名使用

正文首次出现建议写：

```text
Realm Ruckus
```

全大写主要用于Logo：

```text
REALM RUCKUS
```

不得随意写成：

```text
RealmRuckus
Realm-Ruckus
Ruckus Realm
Realms Ruckus
```

除非品牌名称正式变更。

## 十一、文件命名

建议：

```text
Realm Ruckus-Logo-Primary-RGB.png
Realm Ruckus-Logo-Primary-CMYK.pdf
Realm Ruckus-Logo-White.svg
Realm Ruckus-Logo-Black.svg
Realm Ruckus-Icon-Mark.png
```

版本变化不得覆盖旧资产，应保留版本号或归档记录。
